# Bank Account Manager

A simple C++ application that simulates basic bank account operations, including deposits, withdrawals, balance checking, and file-based data persistence.

> ⚠️ **Work in progress** — features and functionality are still being developed.

---

## Features

- Create a bank account with a holder name and account number
- Deposit and withdraw funds with input validation
- View current balance and full account info
- Save and load account data to/from a file
- Exception handling for invalid operations (negative amounts, insufficient funds, file errors)

---
